# Retail Sales SQL Analysis

## Objective
Analyze sales and profit performance using SQL queries.

## Dataset
Sales dataset imported into MySQL as `global_sales_data`.

## SQL Tasks Covered
- Basic SELECT and record count verification
- Filtering using WHERE and sorting using ORDER BY
- Aggregations using SUM, AVG, COUNT with GROUP BY
- HAVING clause for filtering grouped data
- BETWEEN for date range filtering
- LIKE for customer name pattern search
- Exporting query outputs to CSV

- Quesries_task 3.sql  (All SQL queries)
